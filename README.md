Medinfo:
a web application designed to help users manage their medications safely and effectively.
project overview
this project aims to assist people in organizing and tracking their medication usage. it provides tools for searching medicines, scheduling doses, and accessing important information about each medication.

key features:

- search functionality for finding specific medications
- track your search history to revisit previously looked up medicines
- save favorite medications for quick access
- personal profile page for managing your account settings
- interactive calendar for scheduling when to take each medication
- detailed medication pages showing uses, warnings, and user feedback
- secure authentication system with login and signup
- password recovery option for forgotten passwords

pages included:

- homepage - main search interface with history and favorites (dominik)
- medication detail page - comprehensive information including uses, warnings, and user reviews (dominik)
- calendar page - visual schedule for tracking medication intake times(luis)
- add to schedule page - adding medicines to calender (luis)
- profile page - personal account management(luis)
- signup page - new user registration(dominik)
- login page - existing user authentication(dominik)
- forgot password page - account recovery(luis)
- reset password page- account recovery (luis)
- signup form - collect user information (luis)

purpose
the goal is to make medication management simpler and safer by keeping all essential information and scheduling tools in one accessible place.
